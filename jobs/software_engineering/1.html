Software Engineer - Microsoft Power Platform Developer, United States


Organization: Bixal
Country: United States
Office: Bixal in

What will you do ?


As a Software Engineer - Microsoft Power Platform Developer, you will join a diverse team of software architects, engineers and developers creating innovative solutions positively impacting client operations and employee experiences. You will develop custom PowerApps, and SharePoint based solutions supporting knowledge management, organizational learning, human capital planning, employee engagement and collaboration.


Responsibilities


  • Assist in the analysis of client requirements to build SharePoint cloud solutions.

  • Design solutions based on established patterns and best practices for SharePoint Online integration with other Microsoft 365 products.

  • Develop Power Platform applications and solutions (Power Apps, Power Automate, Power BI).

  • Build SharePoint Online sites using WebParts to create visually appealing pages.

  • Develop and customize software with Visual Studio for SharePoint Online, Microsoft 365, Azure using C#, ASP.NET, HTML5 CSS3 and JavaScript.

  • Set up testing environments and demo sites.

  • Familiarity with DevOps and source control.

  • Perform code review and quality assurance checks.


Qualifications


  • Bachelor's degree in Computer Science, Information Systems, or other field with equivalent work experience in information systems.

  • 3+ years developing SharePoint business applications.

  • Experience building customizations and applications using Microsoft Power Platform, (Power Apps, Power Automate, PowerBI).

  • Experience working with client-side technologies like HTML5, CSS3, JavaScript, JQuery.

  • Experience developing client-side web parts and application customizers using SPFx.

  • Experience with Azure DevOps source control using Git.

  • Experience with SharePoint administration and governance.

  • Must be able to obtain a Public Trust clearance.


What will be your salary ?


$94,000 - $116,000 a year
